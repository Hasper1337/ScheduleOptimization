# WinForms: оптимизация расписания занятий

Проект решает задачу минимизации количества учебных дней в неделе при ограничениях:

- 6 дней, 4 пары в день;
- 13 занятий: 4 лекции, 7 практик, 2 лабораторные;
- в одном слоте не более одной пары;
- без окон: если день используется, занятия занимают пары 1..k;
- порядок внутри дня: лекции -> практики -> лабораторные;
- не более 2 лекций в день.

Открыть `UniversityScheduleOptimization.csproj` в Visual Studio на Windows и запустить.
